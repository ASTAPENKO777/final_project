# Tournament FastAPI


## Features
- Users: create, read
- Teams: create, read
- Tournament: create, read
- Result: create, read, read best
- WebSocket Chat

## Tech stack
- FastAPI
- SQLALChemy ORM
- Pydantic
- SQLite

## Install
```
pip install -r requirements.txt
```

Create DB example - env_example

## Run 
```
uvicorn app:app --reload
```
